# GenRichi

GenRichi is a bioinformatics project focused on RNA-Seq and NGS data analysis.
The goal is to build reproducible, scalable pipelines for quality control,
alignment, quantification, and differential expression analysis.

## Project Structure
- data/        : Raw and processed data
- scripts/     : Custom scripts (Python / R / Bash)
- workflow/    : Pipelines (Snakemake)
- results/     : Output files and reports
- docs/        : Documentation and notes

## Tools
- FastQC
- Samtools
- Bedtools
- Python (Biopython, Pandas)
- R (DESeq2, edgeR)
- Snakemake
